# SQL Analysis Queries

This folder contains all SQL queries used to analyze customer shopping behavior.

Each file answers a specific business question.

## Query Index
- Q1: Total revenue by gender
- Q2: Discount vs average spend
- Q3: Top rated products
- Q4: Shipping type analysis
- Q5: Subscription analysis
- Q6: Discounted items analysis
- Q7: Age group analysis
- Q8: Top items by category
- Q9: Purchase frequency analysis
- Q10: Top revenue-generating locations
